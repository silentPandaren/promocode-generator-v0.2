# Генератор промокодов

Веб-приложение для генерации уникальных промокодов с возможностью экспорта в TXT и CSV форматы.

## Возможности

- Генерация уникальных промокодов
- Настройка количества и длины промокодов
- Экспорт в TXT формат
- Экспорт в CSV формат
- Использование только латинских букв и цифр

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Деплой

Проект готов к деплою на Vercel, Netlify или другие платформы, поддерживающие Next.js.
